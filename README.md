## Installation

To get started with this project, follow these steps:

1. **Clone the repository**:

   ```sh
   git clone https://github.com/skiiyuru/home-ground.git
   cd home-ground
   ```

2. **Install dependencies**:

   ```sh
   npm install
   ```

3. **Run the development server**:

   ```sh
   npm run dev
   ```

4. **Build the project**:

   ```sh
   npm run build
   ```

5. **Preview the build**:
   ```sh
   npm run preview
   ```
